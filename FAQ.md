# FAQ — частые вопросы и решения

Собрано из реальных проблем, встреченных при настройке. Если вашей проблемы здесь нет — создайте Issue в репозитории.

---

## Установка

**«python не является внутренней командой» (Windows)**
Переустановите Python с сайта python.org и обязательно поставьте галочку **«Add python.exe to PATH»** в первом окне установщика.

**`pip install -r requirements.txt` падает с ошибкой**
1. Попробуйте `python -m pip install -r requirements.txt`.
2. Убедитесь, что скачали свежий requirements.txt: в старых версиях был пакет OCR, несовместимый с Python 3.13, из-за него отменялась вся установка. С версии 1.0.0.3 такого нет.

**`No module named tkinter` (Linux)**
`sudo apt install python3-tk`

## Локальный ИИ (Ollama)

**«Локальный ИИ (Ollama) не найден», хотя Ollama установлен**
Проверьте по порядку:
1. Ollama запущен? На Windows — иконка ламы в трее возле часов; если нет — запустите из «Пуск». Linux: `ollama serve`.
2. Модель скачана? Установка Ollama её НЕ включает. Выполните `ollama list` — если пусто: `ollama pull qwen2.5:7b`.
3. Откройте в браузере http://localhost:11434 — должно быть «Ollama is running».
4. Перезапустите архиватор.

**Выжимка короткая или странная**
Возьмите модель побольше под вашу оперативную память: 8 ГБ — `llama3.2:3b`, 8–16 ГБ — `qwen2.5:7b`, 16+ ГБ — `qwen2.5:14b`.

**На кадрах нет обводки/стрелки**
Для обводки нужна отдельная vision-модель: `ollama pull minicpm-v`. И честно: локальные модели находят элемент не на каждом кадре — где модель не уверена, кадр сохраняется без разметки (причина пишется в журнал).

## Скачивание

**Ошибка при скачивании видео**
Сайты часто меняются — обновите загрузчик: `pip install -U yt-dlp`.

**Instagram не отдаёт пост**
По порядку: 1) `pip install -U instaloader yt-dlp`; 2) войдите в instagram.com в Chrome на этом компьютере (сработают куки браузера); 3) `instaloader --login ВАШ_ЛОГИН` и укажите логин в программе.

**Telegram не скачивается**
Работают только ссылки на посты публичных каналов вида `https://t.me/канал/123`.

## Доступ с телефона

**Страница не открывается — чек-лист по частоте причин:**
1. **VPN на компьютере в режиме «Глобальный»** (Clash-клиенты: FlClashX и подобные) — переключите на «По правилам». В других VPN включите «Разрешить локальную сеть» / LAN traffic / split tunneling.
2. **Брандмауэр Windows.** Сделайте сеть «Частной» (Параметры → Wi-Fi → ваша сеть → Частная) и разрешите порт: в PowerShell от администратора
   `netsh advfirewall firewall add rule name="Archiver 8080" dir=in action=allow protocol=TCP localport=8080`
3. **Разные сети.** Телефон должен быть в той же Wi-Fi, что и компьютер (не на мобильном интернете). Адрес вводить с `http://`, не https.
4. **Не тот адрес.** Сервер печатает все адреса компьютера — пробуйте начинающиеся с 192.168 или 10.
5. Если сервер написал «РЕЖИМ МАКСИМАЛЬНОЙ ЗАЩИТЫ (обнаружен Tailscale)» — адрес 100.x.x.x открывается только с телефона с установленным и включённым Tailscale (тот же аккаунт).

**Забыл/хочу сменить код доступа**
Удалите файл `web_secret.txt` рядом с программой и перезапустите сервер — создастся новый код.

**Доступ не из дома**
Только через Tailscale (бесплатно, шифрование): установите на ПК и телефон, войдите в один аккаунт — сервер сам перейдёт в режим максимальной защиты. Порт 8080 через роутер не пробрасывать никогда.

**Tailscale + VPN для обхода блокировок: что нужно знать**
- На **телефоне** может работать только один VPN одновременно (ограничение iOS/Android). Если для мобильного интернета вам постоянно нужен обходной VPN — Tailscale вне дома включить не получится. Рабочий компромисс: дома по Wi-Fi всё работает в максимальной защите, вне дома — без доступа к серверу.
- На **компьютере** Clash-клиенты (FlClashX и подобные) дружат с Tailscale, если добавить в правила (троеточие на подписке → «Переопределить») в самый верх: `IP-CIDR,100.64.0.0/10,DIRECT,no-resolve` — и держать режим «По правилам», не «Глобальный».
- Продвинутый вариант для тех, у кого мобильная сеть пропускает Tailscale: настроить домашний ПК как exit node (`tailscale up --advertise-exit-node` + разрешить в админке Tailscale) — тогда вне дома телефону достаточно одного Tailscale: и интернет пойдёт через домашний компьютер с его VPN, и сервер будет доступен.

**Как сменить тему оформления на телефоне**
Кнопки Авто / Светлая / Тёмная под заголовком страницы (с версии 1.0.0.7). Выбор запоминается на устройстве.

## Разное

**Программа «висит» во время обработки**
Это нормально: расшифровка и ИИ-анализ идут минутами — смотрите журнал, там виден каждый этап и прогресс.

**Как откатиться на старую версию**
В окне программы: выпадающий список версий → «Установить версию» → перезапустить. Код всех версий лежит в папке `versions/`, полный бэкап — кнопка «💾 Бэкап».

**Кракозябры в именах файлов после распаковки архива**
Распаковывайте архивы released-версий штатным способом; если распаковщик сломал русские имена — удалите битые файлы и скачайте нормальные из репозитория. В свежих версиях все служебные файлы названы латиницей.
