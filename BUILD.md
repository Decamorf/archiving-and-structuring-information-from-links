# Как собрать устанавливаемое приложение

Приложение уже работает как окно и само докачивает модель распознавания
речи при первом запуске. Ниже — как превратить его в обычный установщик.
Собирать нужно НА ТОЙ ЖЕ системе, под которую делаете дистрибутив.

## Windows (.exe установщик)

Нужно один раз: Python 3.11+ (с галочкой Add to PATH) и программа
**Inno Setup** (бесплатно, jrsoftware.org/isdl.php).

1. Откройте командную строку в папке проекта и выполните:
   ```
   pip install -r requirements.txt
   pip install pyinstaller
   pyinstaller build_windows.spec
   ```
   Появится папка `dist\Архиватор\` с программой (уже запускается:
   `dist\Архиватор\Архиватор.exe`).
2. Откройте файл `installer_windows.iss` в Inno Setup → нажмите
   **Compile** (или Build → Compile).
3. Готово: в папке `Output\` появится **Архиватор_setup.exe** — обычный
   установщик. Пользователь запускает его, жмёт «Далее», получает иконку
   на рабочем столе и в меню Пуск. Удаление — через «Установка и удаление
   программ», как у любой программы.

Иконка: положите файл `icon.ico` рядом со spec (любой конвертер PNG→ICO).
Если иконки нет — удалите строку `icon='icon.ico'` из build_windows.spec.

## Linux (AppImage — двойной клик на любом дистрибутиве)

Нужно: Python 3.11+, wget, интернет.

```
bash build_linux.sh
```

Получится файл **Архиватор-1.2.0-x86_64.AppImage**. Пользователь делает
его исполняемым (правая кнопка → Свойства → Разрешить запуск, или
`chmod +x`) и запускает двойным кликом. Ничего устанавливать не нужно.

## Что произойдёт у пользователя при первом запуске

Программа откроется как окно и в журнале напишет «Готовлю компоненты…»,
скачает модель распознавания речи (~500 МБ, один раз, нужен интернет) и
покажет статус: распознавание речи ✅, сжатие видео ✅, умный ИИ — если
установлен Ollama. Дальше работает офлайн.

Ollama (для умных конспектов) остаётся отдельной установкой по желанию —
это отдельная программа с собственными обновлениями, прятать её внутрь
неправильно. Ссылка и одна команда — в ИНСТРУКЦИИ.

## Автообновление (следующий шаг, версия 1.3.0)

Планируется: приложение проверяет GitHub Releases и предлагает обновиться
одной кнопкой. Ложится на вашу систему тегов v1.2.0, v1.2.1 и т.д.
