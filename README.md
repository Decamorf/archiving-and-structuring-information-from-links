# Архиватор ссылок → Markdown

Локальное приложение (Windows/Linux): принимает ссылку на видео, пост
Telegram/Instagram или статью и сохраняет структурированную заметку .md
с конспектом, скриншотами, командами и ссылками. Всё работает локально
(Whisper + Ollama), раскладка по темам, перевод на любой язык.

- Установка и использование: см. **ИНСТРУКЦИЯ.md**
- Частые проблемы и решения: **FAQ.md**
- История версий: **CHANGELOG.md**
- История версий: вкладка Releases/Tags и папка `versions/`
