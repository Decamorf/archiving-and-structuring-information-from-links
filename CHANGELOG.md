# История версий (Changelog)

Все заметные изменения проекта. Формат: версия — что добавлено/исправлено.
Проект следует Semantic Versioning с версии 1.1.0.

---

## 1.7.1 — исправления языка и логов
- Исправлено сваливание локальной модели в китайские иероглифы: жёсткое
  требование русского языка в промптах + очистка иероглифов из названий
  файлов, тегов, тем и конспекта (страховка на выходе).
- Перевод теперь применяется ко всему конспекту и тоже очищается от
  иероглифов.
- Логи в Telegram-боте теперь полностью совпадают с логами на ПК
  (раньше в чат уходили только ключевые этапы).

## 1.7.0 — поиск и дашборд по архиву
- Кнопка «🔍 Архив»: поиск по заголовкам, темам, тегам и тексту всех
  заметок в реальном времени; двойной клик открывает заметку.
- Дашборд: всего заметок, разбивка по темам, частые теги, последние
  сохранённые.
- Исправлена сборка .exe: модель распознавания речи (Whisper VAD) теперь
  включается в установщик; расшифровка не падает, если её нет.
- В собранной версии убрана нерабочая попытка читать куки браузера для
  Instagram (ошибка DPAPI).

## 1.6.0 — база знаний
- Теги: ИИ проставляет каждой заметке 3-6 тегов.
- Совместимость с Obsidian: YAML-заголовок (title, source, tags, date)
  в начале каждой заметки.
- Дедупликация: повторная ссылка не обрабатывается заново, открывается
  прежняя заметка (метки отслеживания игнорируются).
- Плейлисты и каналы разворачиваются в отдельные видео и встают в очередь
  (до 50 за раз).

## 1.5.0 — автообновление, иконка, безопасность
- Автопроверка новых версий на GitHub Releases при запуске и по кнопке
  «Обновления»; скачивание установщика по подтверждению.
- Иконка приложения для .exe.
- Файл SECURITY.md (политика сообщения об уязвимостях).

## 1.4.1 — устойчивость Telegram-бота
- Повторные попытки при обрыве связи с Telegram.
- Проверка лимита файла (50 МБ) и увеличенный таймаут отправки.

## 1.4.0 — Telegram-бот
- Управление из Telegram: ссылка боту → обработка на ПК → результат
  архивом .zip (заметка + фото/кадры), логи дублируются в чат.
- Доступ только по вашему Telegram ID; токен хранится локально.

## 1.3.3 — аудит безопасности
- Закрыт SSRF при скачивании фото Telegram (шло мимо проверки адреса).
- Полный повторный аудит: инъекции, пути, авторизация, ReDoS — чисто.

## 1.3.2 — данные в папке пользователя
- При установке в Program Files данные (Архив, журналы, бэкапы) пишутся
  в C:/Users/<имя>/Архиватор ссылок, а не рядом с программой.

## 1.3.1 — точность SSRF-защиты
- Белый список доверенных CDN (twimg, cdninstagram и др.) — фото
  перестали ложно блокироваться из-за VPN/DNS.
- Диагностика причины блокировки в журнале.

## 1.3.0 — журнал
- Кнопка «Копировать журнал» и запись журнала в файл в папке logs.

## 1.2.1 — фиксы Twitter
- Публичные CDN больше не блокируются SSRF-защитой при капризах VPN.
- Очистка иероглифов из названий тем/папок.

## 1.2.0 — обычное приложение
- Запуск как окно без консоли; при первом запуске автоматически
  докачивается модель распознавания речи с прогрессом.
- Файлы сборки установщика (Windows .exe через PyInstaller + Inno Setup,
  Linux AppImage).

## 1.1.1 — правки после внешнего аудита
- SSRF-защита в download_file, лимиты размера скачиваемого, защита
  бэкапа от ZIP-slip, маскировка путей в веб-логе.
- Зависимости обновлены до актуальных версий.

## 1.1.0 — Twitter/X и SemVer
- Поддержка постов Twitter/X (текст, фото, видео с расшифровкой) через
  публичные зеркала.
- Переход на Semantic Versioning.

## Ранние версии (1.0.0.1 – 1.0.0.9)
- **1.0.0.9** — точная проверка доменов вместо подстрок (находка CodeQL).
- **1.0.0.8** — SSRF через редиректы + лимит размера страниц.
- **1.0.0.7** — SSRF-защита переходов по ссылкам; выбор темы в мобильной
  версии.
- **1.0.0.6** — вход в веб без кода в адресной строке; список всех адресов
  сервера; подсказка про VPN.
- **1.0.0.5** — защищённый веб-доступ с телефона: код доступа, защита от
  перебора, авто-режим Tailscale (максимальная защита).
- **1.0.0.4** — очередь ссылок: обработка по одной, статусы в таблице.
- **1.0.0.3** — исправлена установка на Python 3.13 (OCR стал
  необязательным).
- **1.0.0.2** — обводка элементов на скриншотах; настоящий OCR (RapidOCR)
  для текста из видеоряда вместо vision-модели.
- **1.0.0.1** — резервные копии и система версий с откатом; точные
  скриншоты со стрелкой; проход по ссылкам с выжимкой; сжатая копия видео;
  перевод на любой язык; текст из видеоряда; иностранные слова латиницей.

## Версия 1 (эталон)
- Базовое приложение: ссылка → структурированная заметка .md. Поддержка
  видео (YouTube и др. через yt-dlp + Whisper), постов Telegram, Instagram,
  статей. Раскладка по темам-папкам. Умные конспекты через локальный
  Ollama, скриншоты, определение тем. Веб-версия для телефона.
